# Ghana-Premier-League-2025-26-Football-Performance-Intelligence-System
## 📊 Overview
A data analytics project that transforms Ghana Premier League team statistics into actionable football intelligence.

## 🎯 Objectives
- Identify title contenders and relegation risks
- Analyze home vs away performance dependency
- Detect late-game defensive weaknesses
- Determine key drivers of winning
- Measure overperformance and underperformance

## 📦 Dataset
Team-level aggregated statistics (2025/26 GPL season)

## 🧠 Key Insights
- Home advantage is a major factor in GPL performance
- Late-game (81–90 min) is the highest vulnerability period
- Winning is driven more by consistency than raw wins
- Several top teams are underperforming expectations
